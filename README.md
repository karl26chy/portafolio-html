# 🚀 Portafolio Profesional - Carlos Charris

¡Bienvenido a mi portafolio personal! Este proyecto es una vitrina de mi trayectoria como **Desarrollador en formación**, donde integro mis conocimientos en **Administración de Empresas** con soluciones tecnológicas creativas y funcionales.

## 📋 Sobre el Proyecto

Este portafolio ha sido diseñado bajo una estética **minimalista y cinemática**, enfocándose en la experiencia de usuario y la limpieza visual. El objetivo es presentar mis habilidades en desarrollo web, programación con Python y mi faceta creativa en la producción musical.

### Características principales:
- **Diseño Responsive:** Optimizado para dispositivos móviles, tablets y escritorio (Mobile First).
- **Menú Hamburguesa:** Implementación personalizada con animaciones suaves en CSS y lógica de JavaScript.
- **Galería Dinámica:** Sección de proyectos y una página dedicada a "Mis Mascotas" que utiliza CSS Grid y efectos de hover.
- **Identidad Visual:** Uso estratégico de filtros de escala de grises y tipografía elegante (*Playfair Display* e *Inter*).

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica para mejorar el SEO y la accesibilidad.
* **CSS3:** Estilos personalizados, Flexbox, CSS Grid y animaciones (`keyframes`, `transitions`).
* **JavaScript (Vanilla):** Lógica para la interactividad del menú, animaciones de desplazamiento y manipulación del DOM.
* **Python:** Mención en proyectos de lógica y gestión (como el sistema de inventarios).
* **FL Studio:** Herramienta clave para mis proyectos de producción musical integrados.
